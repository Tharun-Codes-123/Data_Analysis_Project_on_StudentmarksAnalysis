#  🚀Student Result Analysis Project

## 📖Description
This project analyzes student performance data using Python libraries like NumPy, Pandas, Matplotlib, and Seaborn.  
It helps identify trends, patterns, and insights such as subject-wise performance, study hours impact, and overall grading distribution.

## ✨Features
- Data Cleaning using pandas library
- Visualization using Matplotlib and Seaborn
- Performance comparision across subjects
- Insights based on weekly study hours

## 🛠️Tech Stack
here i used various tools/libraries and the dataset i used to Analyse students scores is from Kaggle
  - Jupyter Notebook
  - Python
- Python libaries
   - Numpy
   - Pandas
   - Matplotlib
   - Seaborn

3. Students who studied more than 10 hours Weekly obatained the most marks
